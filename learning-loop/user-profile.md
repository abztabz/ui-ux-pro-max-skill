# Context — who we're working with and for

Durable model of the people and brand. This is the highest-leverage Context (per M-5):
it pre-loads default decisions on every cycle. Update as you learn more; don't let it rot.

## The owner (the person giving instructions)
- **Non-technical.** No coding knowledge; doesn't read code. Judges work by how it *looks
  and feels*, via preview screenshots.
- **Design-led & iterative.** Moves in fast, small increments ("make it navy", "menu bar",
  "now the contact page") rather than one big spec. Feeds content and references in
  **chunks** — pasted text, screenshots, links — over many turns.
- **Decisive & trusting.** Approves crisply ("Lets go", "push it", "He loves it"). Gives
  the green light fast once they've seen a preview.
- **Budget = $0, by constraint.** Will only use free tooling. Already on: GitHub Pages →
  Netlify hosting, Calendly (free), Netlify Forms (free). A custom domain (~$12/yr) was
  the one paid item discussed and deferred.
- **Building for a client (Amit), not themselves.** Relays the client's reactions.

**What works with them:** plain language, one decision at a time, a preview *before* you
act, honest flagging of limits and gaps, and doing the literal task asked. **What doesn't:**
jargon, multi-step instructions in one breath, unrequested strategy detours, silent scope
or behaviour changes.

## The brand
- **Aesthetic:** editorial, authoritative, premium. Not playful, not flashy.
- **Palette:** deep navy `#16294D` (ink + dark surfaces) + gold-bronze accent `#9A6A3C`;
  warm ivory `#F7F5EF` background. Gold `#E3A92E` for the "Book Consultation" CTA.
- **Type:** Playfair Display (headings) + Inter (body).
- **Voice:** structured, clear, systems-oriented; "no motivational filler."
- **Reference site** the user is matching: `lovely-monstera-d4ae19.netlify.app` (navy +
  gold, gold CTA button, 6-page structure incl. an interactive assessment).
