# Amit Shrestha — Leadership Strategist (Website)

A complete one-page website. It's a single file (`index.html`) in this `/docs`
folder — no build step, no dependencies, no monthly cost.

## How to put it online for FREE on GitHub Pages (no coding)

This uses GitHub's built-in "deploy from a branch" hosting — GitHub serves the
files directly, so there's nothing to build and no workflow to fail.

1. Open: **https://github.com/abztabz/ui-ux-pro-max-skill/settings/pages**
2. Under **"Build and deployment"**:
   - **Source:** choose **"Deploy from a branch"**
   - **Branch:** choose **`claude/amit-leadership-coach-site-jjlvx5`** and folder **`/docs`**
   - Click **Save**
3. Wait ~1 minute, then refresh the page. GitHub shows the live link at the top.

Your site will be live at:
**https://abztabz.github.io/ui-ux-pro-max-skill/**

> Tip: Once the branch is merged into `main`, you can switch the Pages branch to
> `main` so updates publish automatically when you change the site.

## Before you launch — edit these (open `docs/index.html` in any text editor)
- **Email:** search for `hello@amitshrestha.com` and replace with Amit's real email.
- **Booking / Assessment buttons:** the "Request a Consultation" button opens an
  email. To use a free calendar instead, get a **Calendly** link and paste it
  into the `href="..."` of those buttons. Same for "Take the Leadership
  Assessment" (a free **Tally** form link works well).
- Everything else (text, books, testimonials) is already filled in.

## Alternative: Netlify Drop (also free, even faster)
If you'd rather not touch repo settings: go to **https://app.netlify.com/drop**
and drag the `docs` folder onto the page — you get a live link in ~2 minutes.
