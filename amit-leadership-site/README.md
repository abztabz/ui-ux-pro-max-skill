# Amit Shrestha — Leadership Strategist (Website)

A complete one-page website. It's a single file (`index.html`) — no build step, no
dependencies, no monthly cost.

## How to put it online for FREE (no coding)

Pick ONE:

### Option A — Netlify Drop (fastest, ~2 minutes)
1. Go to **https://app.netlify.com/drop**
2. Drag the `amit-leadership-site` folder onto the page.
3. Done — you get a live link instantly (e.g. `amit-shrestha.netlify.app`).
4. Optional: connect a custom domain later (~$12/yr) in Netlify settings.

### Option B — GitHub Pages (free, uses this repo)
1. Push this folder to a GitHub repo (already done if you're reading this here).
2. Repo → **Settings → Pages** → Source: your branch, folder `/amit-leadership-site`.
3. Your site goes live at `https://<username>.github.io/<repo>/`.

## Before you launch — edit these (open `index.html` in any text editor)
- **Email:** search for `hello@amitshrestha.com` and replace with Amit's real email.
- **Booking / Assessment buttons:** the "Request a Consultation" button opens an email.
  To use a free calendar instead, get a **Calendly** link and paste it into the
  `href="..."` of those buttons. Same for "Take the Leadership Assessment"
  (a free **Tally** form link works well).
- Everything else (text, books, testimonials) is already filled in from Amit's content.
